# Windows 需要安装安装的依赖环境
VCForPython27.msi, 直接双击安装即可

ta-lib-0.4.0-msvc.zip, 解包到C盘跟路径即可，注意：解包后的目录名字不能修改

# Python 需要安装的软件包名：
prettytable 2.6.1
pandas 0.23.0  (版本必须是 0.24 之前，不包括0.24版)
ta-lib 0.4.17
pycrypto 2.6.1
schedule 0.6.0

# 手动修正 crpyto 软件包的bug
C:\Python27\Lib\site-packages在这个路径下面有一个文件夹叫做crypto,把它的首字母改成大写，即是Crypto
